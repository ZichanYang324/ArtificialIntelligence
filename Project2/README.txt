For the first and second line to input player type, please only input the numbers;
(human player = 0, random player = 1, minimax player = 2,alpha_beta player = 3)

e.g 
(1)If you want to play minimax player vs. alpha_beta player, you need to input 2 and then input 3 from keyboard.


(2)If you want to play with human player, you also need to input your choice of movement.
 As for human player, if you want to choose the pit [=2], please input the number 2 from keyboard. 
If it is illegal movement, the program will remind you and then you need to input the legal move.
